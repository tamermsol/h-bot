🔧 Fix Background Image Error

Current Error:
"Failed to update background: Could not find the 'background_image_url' column"
Solution: Run a SQL migration to add the missing database column.

Step-by-Step Instructions

Step 1: Open Supabase SQL Editor

Click this button to open your Supabase SQL Editor:

Step 2: Copy the SQL

Click this button to copy the SQL migration code:

✅ SQL copied to clipboard!

The SQL code that will be copied:

-- Add background_image_url to rooms table DO $$ BEGIN IF NOT EXISTS ( SELECT 1 FROM information_schema.columns WHERE table_schema = 'public' AND table_name = 'rooms' AND column_name = 'background_image_url' ) THEN ALTER TABLE public.rooms ADD COLUMN background_image_url TEXT; RAISE NOTICE 'Added background_image_url column to rooms table'; ELSE RAISE NOTICE 'Column background_image_url already exists in rooms table'; END IF; END$$; -- Add background_image_url to homes table DO $$ BEGIN IF NOT EXISTS ( SELECT 1 FROM information_schema.columns WHERE table_schema = 'public' AND table_name = 'homes' AND column_name = 'background_image_url' ) THEN ALTER TABLE public.homes ADD COLUMN background_image_url TEXT; RAISE NOTICE 'Added background_image_url column to homes table'; ELSE RAISE NOTICE 'Column background_image_url already exists in homes table'; END IF; END$$; -- Verify the columns were added SELECT 'rooms' as table_name, column_name, data_type FROM information_schema.columns WHERE table_name = 'rooms' AND column_name = 'background_image_url' UNION ALL SELECT 'homes' as table_name, column_name, data_type FROM information_schema.columns WHERE table_name = 'homes' AND column_name = 'background_image_url';

Step 3: Paste and Run

  1. In the SQL Editor, click "New Query"
  2. Paste the SQL code (Ctrl+V)
  3. Click "Run" or press Ctrl+Enter
  4. Wait for "Success" message

Step 4: Verify Success

You should see output with 2 rows:

table_name | column_name | data_type -----------|-----------------------|---------- rooms | background_image_url | text homes | background_image_url | text

If you see 2 rows, it worked! ✅

Step 5: Test in App

  1. Close and restart your app
  2. Go to any room
  3. Tap settings → Background Image
  4. Try selecting a background
  5. Error should be gone! ✅
✅ Migration Complete!
Now restart your app and test the background image feature.
