## Typography
- **Font Family:** System/Inter stack
- **Heading sizes:** 
  - H1: 60px, weight 800 (hero)
  - H2: 36px, weight 700 (section headings)
  - H3: 24px, weight 600 (feature titles)
- **Body:** 18px, weight 400
- **Small/muted:** 14-16px, weight 400

## Button Styles
- **Primary CTA:** `bg: rgb(59, 130, 246)` (blue-500), white text, `border-radius: 8px`, padding `8px 16px`
- **Secondary:** Transparent/outlined, white text
- **Nav links:** No bg, white text, subtle hover

## Visual Language
- **Dark theme dominant** — near-black backgrounds
- **Glassmorphism cards** — semi-transparent with blur
- **Blue/purple gradient accents** — used across CTAs and highlights
- **Clean, modern, tech-forward** aesthetic
- **Minimal shadows** — relies on transparency and border contrast

## Design Direction Summary
The website uses a **dark, premium tech aesthetic** with electric blue as the primary action color and purple/violet gradients as accents. This is a more sophisticated palette than the current app's Material Blue (#2196F3).

**For the app:** The design agent should use the website's blue (`#3b82f6`) as the primary color base, but since the app is **light-mode only**, adapt the palette:
- Light mode background: warm white (`#F8FAFC`)
- Cards: white with subtle borders
- Primary actions: `#3b82f6` (matching website blue)
- Accents: `#8b5cf6` purple for secondary highlights
- Keep the premium, clean feel from the website
