## Tuya Discussion (09:05 UTC)
- Tim asked how Tuya app does self-discovery
- Tuya uses **BLE advertisements** for device discovery (works on both iOS and Android)
- iOS cannot scan WiFi networks — Apple platform restriction, no public API
- `NEHotspotHelper` can see scan results but requires special Apple entitlement (rarely granted)
- Tasmota devices (ESP8266) don't have BLE hardware, so BLE discovery isn't an option
- ESP32-based devices could support BLE discovery
