### ASC API Lessons
- `appStoreVersionSubmissions` (v1) is deprecated — use `reviewSubmissions` + `reviewSubmissionItems`
- App Privacy / data usages have NO API endpoint — must use ASC web interface
- `appPriceSchedules` needs `included` array with inline `appPrices` referencing price point ID
- iPad Pro 12.9" 3rd gen screenshots (`APP_IPAD_PRO_3GEN_129`) are REQUIRED for submission
- Screenshots with alpha channel → `IMAGE_ALPHA_NOT_ALLOWED` error; use `convert -alpha off PNG24:output.png`
- `contentRightsDeclaration` goes on `/v1/apps/{id}` not `/v1/appInfos/{id}`
